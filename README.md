# ✨ AI Resume Analyzer Pro

An AI-powered tool built with Streamlit that helps analyze resumes and provide actionable insights for job seekers and recruiters.

## 🚀 Features
* **Resume Parsing**: Extracts key details (skills, experience, education) from uploaded resumes.
* **Keyword Matching**: Compares resumes against job descriptions to highlight missing skills.
* **Scoring System**: Generates a compatibility score based on job requirements.
* **Interactive UI**: Simple, premium, and intuitive interface powered by Streamlit.
* **Interview Prep**: Synthesizes behavioral and technical questions based on the active candidate profile.

## 🛠️ Tech Stack
* **Frontend**: Streamlit
* **Backend**: Python (SQLite, Groq AI)
* **Libraries**: NLTK / spaCy (NLP), PyPDF2 / python-docx (Parsing), scikit-learn (Analysis), Plotly (Data Visualization)

## 💻 Installation & Usage

1. **Clone the repository**:
   ```bash
   git clone https://github.com/your-username/Resume-Analyzer.git
   cd Resume-Analyzer
   ```

2. **Install dependencies**:
   ```bash
   pip install -r requirements.txt
   ```

3. **Run the application**:
   ```bash
   streamlit run app.py
   ```
