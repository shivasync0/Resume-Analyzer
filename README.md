📄 AI Resume Analyzer
An AI-powered tool built with Streamlit that helps analyze resumes and provide actionable insights for job seekers and recruiters.

🚀 Features
Resume Parsing: Extracts key details (skills, experience, education) from uploaded resumes.

Keyword Matching: Compares resumes against job descriptions to highlight missing skills.

Scoring System: Generates a compatibility score based on job requirements.

Interactive UI: Simple and intuitive interface powered by Streamlit.

Export Options: Download analysis reports for further use.

🛠️ Tech Stack
Frontend: Streamlit

Backend: Python

Libraries:

NLTK / spaCy for NLP

PyPDF2 / docx for resume parsing

scikit-learn for scoring & analysis
